Open an existing file

Select File → Open (Ctrl-O) and browse to the file you want to open.

By default only XML files are shown but you can also filter for XHTML/HTML files or all files.

If the file references a DTD, XEditNet will attempt to open the file as a valid XML document. If this fails because of validation errors you can attempt to open the document as well-formed. When this happens XEditNet manually validates the document after opening which may take some time for very large documents.
