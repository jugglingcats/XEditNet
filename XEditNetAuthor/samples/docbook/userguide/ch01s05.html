Insert a new element

To insert a new element, select Content → Insert Element (Ctrl+I).

By default XEditNet displays elements that are valid in the current context. To show all elements in the DTD, uncheck Only Show Valid Items (Alt+V).

Elements that are required are shown with a red next to them. If an element is required but more than one option is available, these elements are shown with a green next to them (you must insert at least one of these elements).

Press Return to confirm, or Escape to cancel.

To filter items in the list, enter a partial element name. For example, a will match abbrev, acronym and so on. Enter a space after the name, such as "a " for an exact match.

If the document does not have a DTD, no elements are shown. To create a new element, enter the name of the element in the filter text box and press Return.

Valid elements list

XEditNet shows all elements that are are valid in the current context (that is, at the cursor). However it does not check that the content of the inserted element would be valid. This is intentional as it is the sensible action in most cases; for example, section elements often have required sub-elements which you typically insert after inserting the section. However, it can be a little confusing in some cases; for example, if you select some text or an element that is then invalid in the context of the inserted element.
