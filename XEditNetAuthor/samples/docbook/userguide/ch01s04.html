Select, cut copy and paste text

Use the standard keys such as Shift-RightArrow to select text, or drag select using the mouse. To quickly select the current element contents, select Edit → Select Content (Ctrl+E). Repeat this action to progressively increase the selection upwards to the document element.

Cut, copy and paste using the Edit menu or the standard Ctrl-X, Ctrl-C, Ctrl-V key combinations.

You can also paste text from other applications. XEditNet first looks at the clipboard for content that was copied from XEditNet. It then attempts to parse the data as XML and import into the current document. If that fails but there is text on the clipboard it will paste the raw text into the current document.
