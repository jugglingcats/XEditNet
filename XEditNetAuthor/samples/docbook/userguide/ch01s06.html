Work with attributes

To edit element attributes, select Content → Modify Attributes (Ctrl+L).

XEditNet displays all the existing and possible attributes for the current element. Attributes that are specified have a the icon next to them. Click this icon to remove an attribute.

Validation takes place as you type.

To add a new attribute, click the icon at the top of the window. Note that if the document has a DTD any added attributes not already in the list are automatically invalid.
