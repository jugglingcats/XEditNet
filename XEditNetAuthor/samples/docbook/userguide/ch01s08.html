Hide and show tags

To hide tags select View → Tags Off (Ctrl-K, Ctrl-O). To show them again, select View → Tags On (Ctrl-K, Ctrl-N).

Note that currently XEditNet is intended for use with tags on. The tags off mode is intended as a very simple preview mechanism. Cursor movement and selection is not quite as natural in tags off mode. This is because, when tags are hidden, no information is given about the possible insertion points between two runs of text.
