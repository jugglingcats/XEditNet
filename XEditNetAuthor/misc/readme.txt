The source files in this directory include various utilities from code sample sites.

As far as I know these are licence and copyright free.

PersistWindowState.cs
	Persists the size and location of the main XEditNet Author window.
	See http://www.codeproject.com/csharp/restoreformstate.asp.
	
MultiXmlTextWriter.cs
OutputState.cs
	Allows XSL transforms to output multiple files.
	See http://msdn.microsoft.com/library/en-us/dnexxml/html/xml06162003.asp.
	
MRUManager.cs
	Manages the recently used file list in the File menu.
	See http://www.codeproject.com/csharp/mru.asp.
